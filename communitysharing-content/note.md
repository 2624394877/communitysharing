# IDEA创建maven模块

## maven-archetype-quickstart
- 说明： 自动生成标准的 Maven 项目目录结构

## maven-archetype-site-simple
- 说明： 用于创建带有简单网站功能的基础 Maven 项目，包含简单的网站生成能力

## 两者的区别
- 说明： 
- maven-archetype-quickstart：仅创建最基础的 Java 项目结构
- maven-archetype-site-simple：在基础项目之上增加简单的网站生成功能
