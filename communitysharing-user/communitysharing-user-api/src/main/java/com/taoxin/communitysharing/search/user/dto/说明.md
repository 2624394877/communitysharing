# DTO 实体
- 定义： DTO（Data Transfer Object）是数据传输对象，用于封装数据，通常用于数据传输，比如从数据库中查询数据，然后返回给前端，这时就可以使用 DTO 对象来封装查询结果。
- 与VO实体的区别：
  - DTO（Data Transfer Object）主要用于业务层与表现层之间的数据传输，封装了业务逻辑所需的数据，减少网络传输次数，提高性能。
  - VO（View Object）则是专门用于视图展示的对象，通常是将DTO或其他数据源的数据进行转换，以便于前端页面直接使用，更注重数据的表现形式和格式化。
  - 简单来说，DTO关注的是数据的传输和业务逻辑层面的封装，而VO关注的是如何将数据以合适的形式展现给用户。
